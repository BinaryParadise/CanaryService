# 金丝雀

一个简单的工具平台，只在提示开发测试工作效率。

> [反馈和建议](https://github.com/BinaryParadise/CanaryService/issues)

主要功能：
- 远程配置
- 日志监控
- 设备信息
- Mock数据

## Web

```bash

管理员账号: admin（默认）
密码: admin（默认）
```

## iOS SDK

> 金丝雀只需要登录一次，登录信息存储在keychain中

```bash
pod 'Canary', :configurations => ['Debug']
```
